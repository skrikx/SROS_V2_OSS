sros - SROS V2 local-only operator CLI for governed runs, traces, receipts, and fabric workflows

Usage:
  sros <command> [flags]

Commands:
  init         Initialize local config for SROS v2
  bootstrap    Show bootstrap mode and service boundaries
  doctor       Run local environment and repo integrity checks
  seed         Create a local seed artifact for smoke workflows
  config       Show resolved config and source
  compile      Compile intent through SR8 into canonical run contract
  run          Admit a canonical run contract into SR9 runtime
  plan         Run runtime preflight planning on a canonical run contract
  resume       Resume a runtime session
  pause        Pause a runtime session
  checkpoint   Create a runtime checkpoint record
  rollback     Rollback a runtime session to a checkpoint
  trace        Trace evidence surfaces for inspection and replay
  receipts     Receipt export and closure proof inspection surfaces
  memory       Memory continuity surfaces
  mirror       Mirror witness and drift surfaces
  inspect      Inspect local wiring, readiness, and repository surfaces with operator-friendly snapshots
  status       Show runtime status, persistence posture, and boundary wiring
  tools        Governed tool fabric surfaces
  connectors   Governed connector capability surfaces
  mcp          Governed MCP capability surface
  replay       Replay stored evidence or replay fixtures
  verify       Verify first-run readiness, persistence posture, docs, examples, and showcase surfaces
  release      Local release baseline surfaces
  test         Local smoke and showcase test surfaces
  examples     Example catalog, killer examples, and showcase runner surfaces
  scaffold     Scaffold local artifact and persistence directories

Fast Path:
  sros verify        Check front-door readiness and local surfaces
  sros examples run  Load the example catalog and showcase paths
  sros test smoke    Exercise the real local machine surfaces

Examples:
  sros verify
  sros examples run
  sros trace inspect --input examples/trace/run_trace_min.json

Global Flags:
  --config <path>      Use explicit config file
  --format <text|json> Set output format
  --workspace <path>   Override workspace root
  -h, --help           Show help

Run 'sros help <command>' for more details.
