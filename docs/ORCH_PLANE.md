# ORCH Plane

ORCH owns execution planning hydration, lawful work sequencing, dispatch ordering, checkpoint routing, and artifact coordination.

- Plans are hydrated from admitted runtime sessions plus canonical run contracts.
- Work units are serialized by dependency order and constrained by explicit concurrency rules tied to run risk.
- Ask-mode routes through local approval artifacts only.
- Artifact emission remains coordination-only in W06. Real fabric execution remains deferred to W09.
